# eda-cli (HW03)

Небольшое CLI-приложение для первичного разведочного анализа данных (EDA).
Проект выполнен в рамках HW03.

## Структура проекта

- src/eda_cli/ — исходный код
- tests/ — тесты
- data/ — пример датасета
- pyproject.toml — зависимости и настройки проекта

## Доступные команды

### overview
Выводит общую информацию о датасете.

Пример:
```bash
uv run eda-cli overview data/example.csv


report

Генерирует markdown-отчёт с информацией о качестве данных.

Поддерживаемые параметры:

--out-dir — папка для отчёта

--title — заголовок отчёта

Пример:

uv run eda-cli report data/example.csv --out-dir reports_hw03 --title "HW03 EDA Report"

Качество данных

В проекте реализованы эвристики:

поиск константных колонок

поиск числовых колонок с большим числом нулей

Для каждой эвристики выставляются логические флаги и рассчитывается интегральный показатель качества.

## HTTP API (HW04)

В рамках HW04 реализован HTTP-сервис поверх проекта eda-cli
с использованием FastAPI.

### Запуск сервиса

```bash
uv run uvicorn eda_cli.api:app --reload --port 8000

После запуска документация API доступна по адресу:
http://127.0.0.1:8000/docs

### Реализованные эндпоинты

GET /health
Проверка состояния сервиса.

POST /quality
Оценка качества датасета по переданным параметрам
(количество строк, столбцов, доля пропусков и т.д.).

POST /quality-from-csv
Загрузка CSV-файла и оценка качества датасета
с использованием EDA-ядра.

POST /quality-flags-from-csv
Пользовательский эндпоинт HW04.
Принимает CSV-файл и возвращает полный набор флагов качества,
включая дополнительные эвристики, реализованные в HW03.
